## âš¡ Why Gatsby?

Gatsby was chosen for this portfolio to prioritize **performance and reliability** while keeping the codebase simple and maintainable.

- **Static-first rendering** for fast load times and strong Lighthouse scores
- **SEO-friendly output** ideal for a public-facing personal site
- **Simple deployment on Netlify** with predictable builds and caching
- **Flexible architecture** that allows page content to be composed from reusable section components rather than route-based pages

For a single-scroll portfolio focused on polish and performance, Gatsby provides an excellent balance between developer experience and production quality.
